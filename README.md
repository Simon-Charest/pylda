# zelda-js

## Introduction
A JavaScript [The Legend of Zelda](https://en.wikipedia.org/wiki/The_Legend_of_Zelda) (NES, 1986) clone.

## Prerequisites
- [NGNIX](https://en.wikipedia.org/wiki/Nginx).

## Technologies
- HTML;
- JavaScript;
- [Kaboom](https://kaboomjs.com/).
